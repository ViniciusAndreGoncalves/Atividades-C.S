QUESTÕES SOBRE PROTOTIPAÇÃO DE INTERFACES
Aluno: Vinícius Gonçalves | Data: 13/05/2024
1. O que é o Balsamiq Wireframes? Qual estilo de prototipação ele utiliza?

Resposta: O Balsamiq Wireframes é uma ferramenta de design de interfaces para usuários, que auxilia na criação, de forma ágil de esboços, ou também conhecido como wireframe(protótipo de site, onde cada coisa vai ficar), que serve tanto para aplicativos web como para dispositivos móveis. Ela é amplamente utilizada por designers, desenvolvedores e gerentes de produto para planejar e comunicar a estrutura e o layout de uma interface antes de passar para o desenvolvimento real. O estilo de prototipação é chamado de "low-fidelity", que se parece muito com um desenho 'bruto' feito a mão, justamente para se parecer com um. Isso é devido ao fato de que o software foca na criação do esqueleto do site, ou seja, o que importa é testar a usabilidade e disposição dos elementos do site de forma básica.

2. O que é o figma? Qual estilo de prototipação ele utiliza?

Resposta: O Figma tamébm é uma ferramenta de design de interface para usuários, porém com a vantagem sobre o Balsamiq que é a criação em conjunto(co-op). Geralmente as empresas de software utilizam esse software(via web). O Figma utiliza dois tipos de prototipação, "low-fidelity" e "high-fidelity". Ou seja, além do já explicado anteriormente sobre a prototipagem do "low", a prototipação do "high" significa que o design é posto em mesmo patamar de hierarquia que o esboço do futuro software. Então nessa prototipação está incluído detalhes interativos(animações), ícones bem definídos, fontes e etc. Portanto ele é um software que projeta desde o início, o esqueleto do produto, até o final dele, incluíndo o código que vai ser utilizado.

3. Comente sobre um problema da sociedade que possa ser resolvido por um sistema computacional e porque resolver tal problema é importante.

Resposta: A burocracia estatal. O fato de ainda necessitar de antendimento humano para questões simples é bem frustrante, e pode ser resolvido com a adoção das automações, que inclusive, estão lentamente sendo incluídas. Por exemplo o sistema de busca de contas a pagar da CELESC. Infelizmente ainda se faz necessário o atendimento humano para saber qual é o número da únidade consumidora, mas já é um grande avanço poder ter acesso a esse documento de forma autônoma.
